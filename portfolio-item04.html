Weather Forecast Scraper

A website data harvester

This is a web crawler I created as one of my side projects.

Essentially, this tool can be used to extract contents and data from the weather API to determine daily weather changes.

I utilized the framework of BeautifulSoup to parse the HTML contents of the API. Furthermore, I implemented the HTTP methods to make request calls by leveraging the Python Request library. Additionally, I implemented the python Pandas library to store the data in an excel/CSV file.
